# Bug ID: BUG-002

## Title:
Registration allows invalid email format

## Steps to Reproduce:
1. Open registration page
2. Enter invalid email format (e.g., invalid-email)
3. Enter valid password and confirm password
4. Click Register

## Expected Result:
Error message appears: "Invalid email format"

## Actual Result:
User account is created incorrectly

## Severity:
High

## Priority:
High

## Status:
Open

## Notes:
Tested on Chrome 112, Windows 10
