# Bug ID: BUG-003

## Title:
Profile allows invalid phone number

## Steps to Reproduce:
1. Login to system
2. Navigate to profile page
3. Enter phone number with letters (e.g., abc123)
4. Click Save

## Expected Result:
Error message appears: "Invalid phone number"

## Actual Result:
Profile is updated incorrectly

## Severity:
Medium

## Priority:
Medium

## Status:
Open

## Notes:
Tested on Chrome 112, Windows 10
